## Description

[Nest](https://github.com/nestjs/nest) + [Knex](https://knexjs.org/) test repository.

## Installation

```bash
$ npm install
```

## Preparations

```bash
# docker startup
docker-compose up

# migrations
npx knex migrate:latest
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
